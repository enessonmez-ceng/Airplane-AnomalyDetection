## 📖 Overview
The **Airplane Anomaly Detection** project is a machine learning-based approach to identifying abnormal patterns in aircraft sensor data or flight logs. Detecting anomalies early is crucial for predictive maintenance, ensuring passenger safety, and reducing unexpected downtime for aviation fleets.

This repository contains the data preprocessing scripts, exploratory data analysis (EDA), and machine learning models used to detect these anomalies.

## ✨ Features
* **Data Preprocessing:** Cleaning and normalizing raw flight/sensor data.
* **Exploratory Data Analysis (EDA):** Visualizing sensor trends and identifying potential outliers.
* **Anomaly Detection Models:** Implementation of various detection algorithms (e.g., Isolation Forest, Autoencoders, One-Class SVM).
* **Evaluation Metrics:** Assessing model performance using Precision, Recall, F1-Score, and ROC-AUC.
* **Visualization:** Clear graphs to highlight normal vs. anomalous flight patterns.
